## Telegram реферальный бот

Бот на Python (aiogram 3) для продвижения канала и выплаты вознаграждений за приглашённых пользователей.

### Возможности
- **Индивидуальная пригласительная ссылка** для каждого пользователя при нажатии `/start`.
- **Учёт приглашённых**:
  - За каждого нового пользователя, который вступил в канал по ссылке — пригласившему начисляется **+3 ₽**.
  - Если приглашённый отписался от канала — с баланса пригласившего списывается **3 ₽**.
- **Баланс и вывод**:
  - Команда `/balance` — показать текущий баланс.
  - Команда `/withdraw` — запросить вывод средств (минимум **30 ₽**); заявка уходит администратору.
- **Текстовые условия**:
  - Оплата только за **реальных, не накрученных** пользователей.
  - Вывод средств осуществляется через администратора.
  - В бота может зайти любой пользователь — бот служит для заработка.

### Настройка
1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. В `bot.py` при необходимости измените:
   - `BOT_TOKEN`
   - `CHANNEL_USERNAME`
   - `ADMIN_ID`
3. Добавьте бота в канал `@CHANNEL_USERNAME` как администратора с правами:
   - Управление ссылками-приглашениями (invite links)
   - Просмотр событий участников (желательно все стандартные права администратора).

### Запуск
```bash
python bot.py
```

Бот использует `bot.db` (SQLite) в корне проекта для хранения пользователей, балансов и рефералок.
